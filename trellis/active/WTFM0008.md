# WTFM0008 - Path-prefix portability for generated sites

Taproot serves its wtfm docs site under **`/help/` on the app origin**
(TR00384's concrete serving contract, chosen so ESP0119's same-origin fetch
just works). wtfm has only ever been consumed at an origin root
(`espalier.taproot.test`), and it shows: `data/components.js` hardcodes
root-relative URLs (`"/components"` breadcrumb bases), and nothing documents
how the client runtime, code-block assets, or the `asset` filter behave under
Eleventy's `pathPrefix`.

## Scope

- Audit every URL wtfm generates or consumes for pathPrefix correctness:
  `data/components.js` URLs and breadcrumbs (plus `data/surfaces.js` from
  WTFM0005), the `asset` filter, `inlineSvg`, and any URL the client runtime
  (`wtfm-runtime`, `wtfm-code-block`) constructs at runtime.
- Make the URL base composable: either honor Eleventy's `pathPrefix`
  transparently (preferred — Eleventy's `url` filter/HTML transform handles
  page links; data-helper URLs must not bypass it) or accept an explicit
  base-path option; no hardcoded root-relative strings left behind.
- Emit prefix-absolute URLs in help-document output (WTFM0006 coordinates —
  injected help HTML resolves relative URLs against the *editor's* page, so
  document-relative links are wrong by construction there).
- Verify a fixture site built with `pathPrefix: "/help/"` end-to-end: pages,
  nav, breadcrumbs, assets, and code-block hydration all resolve.
- Document the pattern (an entry in the plugin's options docs plus a short
  serving note) so TR00384 can point at it.

## Notes

- The taproot-controls docs site (no prefix) must keep working byte-identically
  — prefix support is additive, defaulting to the current behavior.
- Consumed by Taproot TR00384; the relative-URL trap it flags for injected
  help HTML is the WTFM0006 coordination point.

## Risks

- Half-prefixed output is worse than none — a page that loads but whose
  runtime assets 404 under `/help/` fails only in the deployed layout. The
  fixture-site verification, not code review, is the gate.
